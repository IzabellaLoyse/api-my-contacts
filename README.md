# API My Contacts

- API desenvolvida para o aprendizado do NodeJS
- Utiliza-se Banco de Dados

# 💡 Guia de instalação

## 🔮 Instalação das Dependências

- Para instalar as dependências necessárias do projeto

Usando npm:

```
npm install
```

Usando Yarn:

```
yarn install
```

## 📌 Rodando o Projeto

Usando npm:

```
npm dev
```

Usando Yarn:

```
yarn dev
```

## ⚙️ Tecnologias Utilizadas

- NodeJS
- PostgreSQL
- Docker
